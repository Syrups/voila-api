## Getting Started

Install the module with: `npm install tenveux`

```js
var tenveux = require('tenveux');
tenveux.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g tenveux
$ tenveux --help
$ tenveux --version
```


```sh
# creates a browser.js
$ grunt browserify
```
